## Run
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Create `.env` with:
```
GROQ_API_KEY=your_key_here
```

Start:
```bash
./run.sh
```

Alternative:
```bash
python main.py
```
